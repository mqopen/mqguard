# mqguard changelog

## v0.1.0

 - Initial version.
 - Implemented alarms:
   - FloodingAlarm
   - TimeoutAlarm
   - RangeAlarm
   - ErrorCodesAlarm
   - DataTypeAlarm
   - PresenceAlarm
 - Application configuration at _.ini_ style config file.
 - Implemented reportes:
   - SocketReporter
   - WebsocketReporter
   - PrintReporter
 - Implemented formatters:
   - JSONFormatter
 - Designed output JSON format.
 - Implemented differential updating.
